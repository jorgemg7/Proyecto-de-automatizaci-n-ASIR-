# 🚀 Mi App AWS con Docker y Flask

Este proyecto es una **aplicación web sencilla en Flask**, desplegada en **AWS EC2** usando **Docker**, como práctica para **AWS SAA** y **DevOps**. Incluye buenas prácticas de despliegue seguro, control de acceso y posibilidad de integración con **S3** para almacenamiento de datos o logs.

---

## 🎯 Objetivo

- Crear una app web funcional con Flask.
- Contenerizarla usando Docker.
- Desplegarla en **AWS EC2** con control seguro mediante **IAM User** y **Security Group**.
- Preparación práctica para certificación **AWS SAA** y experiencia real en DevOps.

---

## 🛠 Tecnologías utilizadas

- **Python 3** + Flask  
- **Docker**  
- **AWS EC2, IAM, Security Group, S3 (opcional)**  
- **Git & GitHub**  
- **AWS CLI**
